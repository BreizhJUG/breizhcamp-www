Notre objectif : proposer une conférence technologique de qualité professionnelle, où les passionnés peuvent enrichir leur boite à outil, étoffer leur réseau, mettre les mains dans le cambouis.

Notre ADN : BreizhCamp est une conférence pour les développeurs, par des développeurs. Le prix d'entrée est symbolique pour permettre à tous les passionnés de venir profiter de ce rendez-vous annuel.

Toutes technologies : BreizhCamp a pour mot d'ordre l'échange et le partage. Avec l'aide des diverses communautés technique Rennaises, notre programme vise à proposer des sujets sur toutes les technologies sans restriction, enrichissants pour tous les publics.
